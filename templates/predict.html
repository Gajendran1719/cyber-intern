Feature Guide

What are these features?

These are network parameters collected from wireless sensor networks including:

  • ID: Node identifier
  • Time: Timestamp of event
  • Is_CH: Is Cluster Head
  • ADV/JOIN/SCH/DATA: Packet counts
  • Rank: Node rank in hierarchy
  • Energy: Expanded energy value
